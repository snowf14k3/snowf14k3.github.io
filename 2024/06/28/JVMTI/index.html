介绍

JVMTI（Java Virtual Machine Tool Interface）是一个由Oracle提供的标准接口，
用于在Java虚拟机（JVM）上创建工具和应用程序。允许开发者在运行时监控和修改JVM的行为，
让JVMTI成为进行性能分析、监控、调试和其他类型工具开发的强大工具。
